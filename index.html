Q&A

Three main differences between Web 1.0, Web 2.0 and Web 3.0:

  • In Web 1.0, the pages are static pages where the user cannot share their ideas or concepts. However, in Web 2.0 and Web 3.0 the pages are dynamic pages allowing users to interact more freely with each other.


  • Web 1.0 is mostly read-only, without being able to post reviews, comments or feedback. Web 2.0 is widly read-write allowing users to participate, collaborate and share information. Web 3.0 is portable and personal including interactive services and “machine-to-machine” interaction.


  • In Web 3.0, computers can interpret information to provide faster and more relevant results to the user. In Web 1.0 and 2.0 informtion is not saved or used and computers do not become more intelligent.


Side Effects of Medicines Use Case

If a user searches for "Side Effects of Medicines":

  • In Web 1.0, static pages will be returned. If the user doesn't find what he is looking for (for example a specific medicine) he will have to insert a new query.

  • In Web 2.0, the user will be able to interact and get advice from other users. He can comment and ask questions on websites for clarification about a specific medicine.

  • In Web 3.0, the most relevant pages that match that user will appear. For example, it could be depending on his previous searches or his location.
